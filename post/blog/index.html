使用Hugo进行个人博客搭建

1 min read
hugo 个人博客搭建

LINUX下进行博客搭建并上传远端服务器

1、需要必备工具git

安装git

sudo apt install git`

2、安装hugo及验证版本

安装hugo。

sudo apt install hugo

查询hugo版本信息。

hugo version

有版本信息即为安装成功。

3、创建个人博客及本地启动

创建个人博客。

hugo new site myblog //myblog可替换为任意名称。

设置个人博客主题，hugo主题下载网址链接。 若直接下载速度较慢，可在gitee上搜索想要的主题并使用git进行下载。(主题文件夹需要放置在博客根目录的themes文件夹下)。

本地启动个人博客。

hugo server -t "主题文件夹的名称" --buildDrafts

启动后通过http://localhost:1313/进行访问。

4、写博客

创建md格式文件进行博客编辑，文件夹名称可自行设置。

hugo new post/blog.md

使用typora或者vscode均可对md文件进行编辑。

5、将个人博客上传到远端服务器

github上创建一个新的仓库以部署博客，仓库名必须为 github昵称.github.io。

部署到远端

hugo --theme=主题名称 --baseUrl="https://github昵称.github.io/" --buildDrafts

此步完成后会在博客根目录下生成一个public文件，进入public文件夹将其上传到仓库

cd public
git init
git add.
git commit -m "博客第一次提交"
git remote add origin https://github.com/github昵称/github昵称.github.io.git //将本地仓库关联到远端
git push -u orgin master //按提示输入github用户名和密码

用https://github昵称.github.io/即可远程访问博客。
